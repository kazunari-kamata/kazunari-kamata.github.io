Featured Project

Infra Lifecycle Portal

インフラ資産のEOLリスク、移行優先度、検証観点、ロールバック手順を整理するための 軽量なライフサイクル管理ツールです。

  • OS / ミドルウェア / ランタイムのEOLリスク検出
  • Markdown形式のレポート生成
  • FastAPI / React / GitHub Actions による構成
Repository

CLI Tool

go2rust-cli

Go ソースコードを Rust ソースコードの雛形へ変換する Rust 製 CLI です。 対応済みの単純な構文を変換し、未対応構文は TODO コメントとして残します。

  • clap による convert サブコマンド
  • regex による初期版の簡易変換
  • 標準出力とファイル出力に対応
  • cargo fmt / clippy / test を実行する CI 構成
Repository

Operations Kit

internal-se-ops-kit

社内SE業務で発生しやすい入退社、アカウント管理、端末管理、インシデント対応、 変更申請を整理するための運用テンプレート集です。

  • 入社・退社対応のチェックリスト
  • アクセスレビュー、変更申請、インシデント報告のテンプレート
  • サンプル資産台帳を検査する Python スクリプト
  • pytest と GitHub Actions による基本的な検証構成
Repository

Sample Repository

go2rust-cli-samples

go2rust-cli の変換入力として使う Go サンプル集です。 対応済み構文と未対応構文の出力を確認しやすい小さな Go ファイルを管理しています。

  • 基本的な main 関数と fmt.Println のサンプル
  • 変数宣言と return 文の確認用コード
  • 未対応の制御構文や関数宣言のサンプル
  • go test ./... を実行する CI 構成
